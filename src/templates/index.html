You're running a BackboneDiorama project.

How to get up and running

  1. Use diorama generators to create your controllers and views.
    • For example, `diorama generateController ControllerName Action1 Action2`
  2. When you're happy with the project structure, compile your code:
    • `diorama compile`
  3. Open up `index.html` and start hacking! Your compiled application will be included automatically.
